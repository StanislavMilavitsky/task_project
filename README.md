  This REST project is built on the basis of three entities (user, project and task). The project used Spring Bolt, Spring Security, Spring JDBC, OAUTH2, Swagger, Maven. The project is built in compliance with the principles of SOLID, KiSS, DRY. PostgreSQL database. The admin can do everything that the user can, plus see all users, see all projects, even deleted, as well as delete projects and users. The technology of building a project according to the MVC pattern. All users have standard CRUD operations, and the Project entity can also be sorted by start date, end date, by naming and search by naming or description. Commeted code in OpenApiConfiguration.java, this is the code for enabling the authorization function via Google. Setting up the connection in application-properties.
